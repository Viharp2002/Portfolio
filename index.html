Time and Work Tracking System

Digital Gujarat Web Portal

"A web portal helping civilians to find and apply for government-granted schemes digitally. Admin or government authority will approve applications. Sportsperson Identifier

"It is a Machine Learning image classifier model to identify the players of different games by uploading a player’s image on portal. Support Vector Machine, Logistic Regression and Random Forest are trained and tested, out of which Logistic Regression achieved the highest accuracy of 88.09%. Worked with HTML, CSS and Javascript for the front end and managed the back end using Python and Flask."

Multiple Disease Predictions
